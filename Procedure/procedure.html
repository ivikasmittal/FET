Procedure


For Drain Characteristics :-


  1. Set an appropriate value of Pinch Off Voltage.
  2. Adjust a constant value for Gate-Source Voltage using the slider.
  3. Change the value of Drain-Source Voltage using the slider.
  4. For each value of Drain-Source Voltage, calculate the corresponding value of Drain Current.
  5. Add each set of values of Drain-Source Voltage and corresponding Drain Current to the observation table using the Add to Table Button.
  6. Plot the graph for the observation table using the Add Data Point and Render Button.


For Transfer Characteristics :-


  1. Set an appropriate constant value of Gate-Source Current.
  2. Set an appropriate value of Pinch Off Voltage.
  3. Adjust a constant value for Gate-Source Voltage using the slider.
  4. Change the value of Drain-Source Voltage using the slider.
  5. For each value of Drain-Source Voltage, calculate the corresponding value of Drain Current.
  6. Add each set of values of Drain-Source Voltage and corresponding Drain Current to the observation table using the Add to Table Button.
  7. Plot the graph for the observation table using the Add Data Point and Render Button.
